# Muse 更新日志

## [1.6.0] - 2026-01-03

### ✨ 重大更新

#### 学习界面优化
- **今日学习记录** - 侧边栏现在记录今天所有学习会话的单词,不仅仅是当前会话
- **历史单词回顾** - 支持点击查看所有今天学过的单词,包括之前学习会话的单词
- **智能导航系统** - "上一个"/"下一个"按钮可在所有已学单词间自由导航
- **学习状态标记** - 清晰标记"认识"/"不认识"状态,紫色标签标识之前学习的单词

#### 设置界面增强
- **快捷键多功能标签** - 使用"/"分隔显示一个快捷键的多个功能
  - "显示答案 / 返回当前学习"
  - "认识 / 下一个"
  - "不认识 / 上一个"
- **快捷键冲突检测** - 自动检测快捷键冲突并显示友好警告提示
- **可视化冲突提示** - 冲突时显示红色警告框,明确指出冲突功能

### 🔧 技术改进
- **本地存储优化** - 使用 localStorage 按日期存储学习记录,自动跨会话保留
- **状态管理优化** - 改进 learnedWords 状态结构,支持三态(null/true/false)
- **组件接口更新** - ShortcutItem 组件支持快捷键冲突状态显示
- **数据持久化** - 学习记录跨多个学习会话持久保存

### 🐛 Bug 修复
- 修复快速按键可能导致的状态混乱问题
- 优化历史单词导航逻辑
- 修复学习会话外单词无法查看的问题

## [1.5.2] - 2026-01-03

#### AI 智能分析优化

- **快捷键设置界面**：在设置页面的快捷键设置中添加了"AI 智能分析"选项

- **移除冗余提示**：去掉了"点击查看释义"文本，避免与"显示答案"功能重复
- **优化界面布局**：移除了操作区下方的快捷键提示文字，界面更简洁
- **移除重复按钮**：去掉了释义区域的 AI 分析按钮，避免功能重复

#### 词库管理优化
- **删除确认弹窗**：为词库删除操作添加了确认对话框，防止误操作
- **视觉警告**：使用红色警告图标和明确的文字提示，增强用户感知

### 🔧 技术改进

## [1.5.1] - 2026-01-03

### ✨ 新特性

#### 快速复习模式
- **新增快速复习功能**：针对想要快速学习的同学，可以不按 SM-2 算法，直接复习所有学过的单词
- **复习模式切换**：在复习页面可以切换"智能复习"（基于艾宾浩斯遗忘曲线）和"快速复习"（所有单词）
- **快速复习数量设置**：在设置页面可以自定义每次快速复习的单词数量（10-100个）
- **优化的复习UI**：复习模式切换器采用边框高亮设计，更加清晰直观

#### 独立关于页面
- **新增独立关于页面**：将关于信息从设置中独立出来，成为单独的页面
- **侧边栏导航**：在侧边栏添加"关于"入口，使用 Info 图标
- **功能特性展示**：展示应用的三大核心特性（智能学习、科学复习、进度追踪）

#### UI/UX 优化
- **统一按钮设计**：所有空状态页面的按钮采用统一的布局和样式
  - 学习页面：词库 + 设置（并排）
  - 复习页面：快速学习 + 刷新（并排）
- **优化按钮配色**：移除渐变色背景，改用纯色 + hover 效果，更加简洁专业
- **开屏动画优化**：Logo 直接作为浮动容器，添加圆角效果，更适合非透明背景的 Logo

### 🐛 Bug 修复

#### 学习与复习体验
- **学习完成界面优化**：移除学习完成后的额外导航按钮，保持界面简洁
- **空状态导航优化**：在"今日学习已完成"时提供前往词库和设置的快捷入口

## [1.5.0] - 2026-01-02

### ✨ 新特性

#### AI 智能学习计划
- **首页智能规划**：在首页新增 AI 学习计划功能，可根据用户目标自动制定个性化学习方案。
- **多维度计划生成**：
  - 支持选择英语水平（初级/中级/高级）
  - 自定义学习目标（考试、工作、日常等）
  - 设置每日可用时间（10-120分钟）
  - 选择目标完成期限（1/3/6个月或不设定）
  - 重点关注领域（商务、学术、旅游等8大类）
- **智能学习建议**：
  - 每日新词与复习数量的科学配比
  - 每周里程碑目标和进度追踪
  - AI 个性化学习策略
  - 推荐适合的词库
- **计划管理**：支持查看已有计划详情，随时重新规划新的学习方案。

### 🐛 Bug 修复

#### 快捷键系统优化
- **修复快捷键生效问题**：解决了用户必须重新设置快捷键才能使其生效的问题。现在修改快捷键设置后立即生效，无需重启。
- **修复语音重复播放**：解决了按下语音播放快捷键或按钮时，语音会多次重复播放的问题。现在播放新语音前会自动取消正在播放的音频。

#### UI/UX 统一优化
- **统一学习与复习界面**：
  - 复习界面移除绿色背景，改为与学习界面一致的白色卡片风格
  - 统一词性标签颜色（学习与复习均使用蓝色）
  - 统一发音按钮样式（使用蓝色主题色）
  - 统一"显示答案"按钮渐变色（蓝紫渐变）
- **复习评分按钮顺序调整**：
  - 按从左到右难度递减：太简单 → 记住了 → 有点难 → 忘记了
  - 符合用户从易到难的操作习惯
- **去除视觉干扰**：
  - 移除复习卡片右上角的待复习数量红色徽章
  - 创建 AI 学习计划后，隐藏原有的通用学习建议，避免信息冗余
- **目标完成日期卡片化**：将日期选择器改为4个卡片式按钮（不设定/1个月/3个月/6个月），操作更直观。

### 🔧 技术改进
- **优化快捷键 Hook**：使用 ref 存储最新的快捷键配置，避免不必要的重复绑定，提升性能。
- **类型安全增强**：为学习计划相关功能添加完整的 TypeScript 类型定义。

## [1.4.4] - 2026-01-02

### ✨ 新特性

#### 安全性与 Web 端优化
- **新增内容安全策略 (CSP)**：在 `index.html` 中配置了严格的 CSP，防止 XSS 攻击并限制非授权的数据连接。
- **新增 Web App Manifest**：添加了 `manifest.json`，提升 Web 端作为应用运行的合规性与安全性。
- **优化引用策略**：设置 `Referrer-Policy` 为 `strict-origin-when-cross-origin`，保护用户隐私。
- **完善安全元数据**：添加了 `robots.txt` 和 `.well-known/security.txt`，提升站点的专业性与安全性评分。

#### 词库管理功能增强
- **新增导出功能**：支持将词库导出为标准 JSON 格式文件，包含完整的单词释义、例句、音标等数据。
- **导入格式升级**：
  - 导入功能现在支持完整的单词对象格式，能够完美还原导出的词库数据。
  - 保持对旧版 `{"单词": 优先级}` 格式的兼容。
  - 优化了导入界面的格式说明和错误提示。

## [1.2.0] - 2025-01-01

### ✨ 新特性

#### AI无缝集成到学习界面
- **移除独立的AI助手按钮**：不再需要手动点击"AI智能助手"按钮
- **自动加载AI内容**：当显示单词答案时，AI会自动在后台分析并生成内容
- **融合式展示**：AI生成的内容直接嵌入到单词释义区域中，与字典释义无缝融合

#### AI增强内容
1. **AI深度解释**
   - 基本含义：简洁的中文解释
   - 详细解释：包括词源、引申义、语境用法
   - 用法要点：使用场景和搭配建议
   - 常见错误：帮助避免常见误用

2. **AI智能例句**
   - 在字典例句后自动添加AI生成的例句
   - 每个例句包含英文原句和中文翻译
   - 例句更贴合实际使用场景
   - 支持一键复制例句

3. **深度理解**
   - 基于AI详细解释的扩展说明
   - 帮助更深入地理解单词用法

#### 视觉优化
- 使用渐变色背景区分AI生成内容（紫色-蓝色渐变）
- AI内容带有 ✨ Sparkles 图标标识
- 加载状态显示友好的"AI正在分析..."提示
- AI例句采用独特的紫色-粉色渐变背景

### 🎯 用户体验改进
- **零感知AI调用**：用户无需主动触发，AI自动工作
- **内容融合**：不再有"单词释义"和"AI助手"的界面分离感
- **智能缓存**：切换单词时自动清理旧内容
- **优雅降级**：当AI服务未配置或失败时，不影响基础学习功能

### 🔧 技术改进
- 移除 `showAIAssistant` 状态管理
- 移除 `AIAssistant` 组件依赖
- 新增 `aiExamples`、`aiExplanation`、`aiMemoryTip` 状态
- 使用 `useEffect` 监听 `showAnswer` 自动触发AI内容加载
- 并行请求AI服务以提升响应速度

### 📝 代码变更
- 文件：`src/pages/Learn.tsx`
  - 重构AI内容加载逻辑
  - 更新释义区域UI布局
  - 集成AI内容到单词卡片中
