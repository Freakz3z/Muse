# Muse 更新日志

## [1.6.3] - 2026-01-07

新功能:

- **字母连线游戏** - 全新单词连线游戏模式
  - 5×5字母网格,点击相邻字母组成单词
  - AI智能生成关卡,每次游戏都是新挑战
  - 2分钟倒计时,考验单词拼写能力
  - 游戏结束自动保存单词到个人单词库

- **游戏中心** - 统一的游戏管理界面
  - 游戏统计面板(学会单词数、完成游戏数、总得分)
  - 个人单词库管理-保存游戏中找到的单词
  - 多游戏模式支持,未来可扩展更多游戏

优化:

- **个人页面优化** - 移除不可用功能,简化界面
  - 移除"添加邮箱"功能
  - 移除"兴趣标签"功能
  - 保留核心学习统计和信息展示

技术改进:

- 新增LetterLinkGameEngine游戏引擎
- 新增LetterLinkAIService AI生成服务
- 游戏数据本地存储和管理
- 完整的游戏生命周期管理

## [1.6.2] - 2026-01-06

AI Native 核心功能:

- AI双按钮独立设计-通用AI(W键)和个性化AI(E键)完全独立
- 个性化记忆提示-Quiz答错后自动显示基于用户画像的记忆技巧
- AI画像可视化-6维用户画像完整展示(认知风格、记忆特征、学习行为、知识图谱、情感状态、学习目标)
- 自适应学习引擎-基于用户画像的个性化学习路径预测和优化
- 个性化内容生成器-根据学习风格动态生成例句、记忆技巧和释义
- AI学习教练-智能分析学习会话,提供疲劳检测、困难识别、心流优化和动机激励
- 智能单词选择-8维度优先级算法,基于遗忘曲线、准确率、间隔阶段等智能选择测验单词

UI 优化:

- Review页面简化-从4个难度按钮简化为3个,后端智能映射到SM-2算法的4级评分
- Learn侧边栏浮层设计-现代化玻璃态效果,带统计面板的悬浮侧边栏
- 配色系统标准化-完整的CSS变量系统,包含100+设计令牌,统一颜色管理

技术改进:

- 新增ai-core服务模块-包含自适应引擎、画像管理、内容生成器、学习教练
- 完整覆盖所有11个界面快捷键
- AI教练除零修复-所有计算添加安全检查,防止边缘情况错误
