# Muse 更新日志

## [1.6.2] - 2026-01-06

### ✨ 新特性 - AI Native 核心功能
- **AI双按钮独立设计** - Learn页面通用AI(W键)和个性化AI(E键)完全独立,快捷键逻辑清晰可预测
- **个性化记忆提示** - Quiz页面答错后自动显示基于用户画像的个性化记忆技巧
- **AI画像可视化** - 全新的AIProfile页面,6维用户画像完整展示(认知风格、记忆特征、学习行为、知识图谱、情感状态、学习目标)
- **自适应学习引擎** - 基于用户画像的个性化学习路径预测和优化
- **个性化内容生成器** - 根据学习风格动态生成例句、记忆技巧和释义

### 🎨 UI 优化
- **Review页面简化** - 从4个难度按钮简化为3个,后端智能映射到SM-2算法的4级评分
- **双向链接导航** - Statistics和AIProfile页面互相链接,方便用户查看
- **测试页面** - 新增TestAdaptiveEngine和TestContentGenerator页面用于开发验证
- **Learn历史模式提示** - 进入历史浏览时显示明确的快捷键提示横幅
- **Review难词重考提示** - 选择"有点难/忘记"后显示"这个词将在稍后再次出现"

### 🔄 学习流程优化
- **Quiz答错自动复习** - Quiz测验中答错的单词自动加入复习队列,形成学习闭环
- **数据流打通** - Quiz结果直接影响学习记录,错题自动标记为需复习(quality=1)
- **用户反馈增强** - 所有重要操作都有明确的视觉反馈和提示

### 🛠️ 技术改进
- 新增 `ai-core` 服务模块,包含自适应引擎、画像管理、内容生成器
- 新增个性化内容类型定义和辅助工具
- 优化快捷键系统,支持更多AI操作
- 移除废弃的StudyPlanModal和AICoach组件
- README添加Twitter分享徽章和Star历史图表

### 📊 量化改进
- Learn页面AI操作点击次数减少33%
- Review难度选项减少25%
- Quiz页面AI功能覆盖率从0%提升到100%
- 快捷键可预测性大幅提升
- Quiz与Review/Learn数据流完全打通
